# WebSprint 15 – Learn HTML & CSS in 15 Days 🚀

WebSprint 15 is a beginner-friendly web development course designed to help you learn **HTML and CSS basics in just 15 days**.

This repository contains all the **source code, exercises, and projects** used in the course.

---

## 📚 What You Will Learn

✔ HTML Basics  
✔ Page Structure & Tags  
✔ Text Formatting  
✔ Links & Images  
✔ Lists & Tables  
✔ Forms  
✔ CSS Basics  
✔ Inline, Internal & External CSS  
✔ Box Model  
✔ Layout & Styling  
✔ Final Portfolio Project  

---

## 📅 Course Structure
Each folder contains:
- Source Code
- Practice Files
- Mini Exercises
- Project Files (if applicable)

---

## 🎯 Who Is This Course For?

- Absolute beginners
- Students starting web development
- Urdu & English learners
- Anyone who wants to build websites from scratch

---

## 💻 How To Use This Repository

1. Download or clone this repository
2. Open files in VS Code
3. Follow along with the YouTube videos
4. Practice daily
5. Complete exercises and projects

---

## 🏆 Final Project

On Day 15, you will build a complete **Personal Portfolio Website** using HTML & CSS.

This project will help you:
✔ Understand real website structure  
✔ Improve layout skills  
✔ Build confidence  
✔ Add a project to your portfolio
